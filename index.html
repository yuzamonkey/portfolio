About me

I'm a full-time student at the University of Helsinki. After studying mathematics as my major for two years, I switched my major to be computer science in 2021. I will complete my bachelor's degree in the autumn semester of 2022. After that, my plan is to continue to master's studies. In addition, I study part-time at the Sibelius Academy, majoring in viola.


My main focus is to build robust, straightforward and comprehensive programs and I always look forward to working in a developer team. I enjoy being challenged and engaged with projects that require me to work outside my comfort and knowledge set. What I can bring to a team, is continuous adaptation to varying circumstances and responsibility to disclose problems before it's too late. I find these qualities to be essential for a successful developer.


Below, I have programming projects I've made so far, and a few memorable moments from my career as a musician.


Programming

Number Recognition

A program to classify hand written numbers from MNIST database. The classification is done with k-nearest neighbor algorithm. The program is written in Python, and the GUI-library used in this program is PyQt5.

Half Eaten Biscuit

Half Eaten Biscuit is a web app targeted for artists to connect and show their work as individuals and groups. I've used TypeScript to write the code. The main tools for this project are React, GraphQL, MongoDB, Jest and Cypress.

Source code

Bookmark library

Group project from software engineering course. The program is written in Java. During the project, we used the techniques defined in Scrum, such as maintaining a product backlog, week-long sprints and customer meetings.

Source code

Scorewriter

A small program to write, save and play a single line of staff. The user can select between 2 clefs, 3 key signatures and 4 time signatures. The code is written in Python and Tkinter-module is used for GUI.

Source code

Event calendar

A simple web app to create ads for events. The code is written in Python and the main tools are Flask and PostgreSQL.

Music

Freelancer

Since the start of 2018 my main source of income, apart from student benefits, has been playing the viola in a lot of different projects. So far, I've enjoyed my hardwork studying computer science and earning an income from playing, but now I'm looking to work on software as my job.

Conducting

During my time in the military service in the Conscript Band of the Finnish Defence Forces, I got my first touches of conducting an orchestra during the NCO-course. After the intensive eight week period, I was awarded a stipend for my success.

Studio recordings

Studio recordings have given me a chance to play multiple genres of music from mainstream summer pop to military marches. My most memorable recordings have been for a movie Eila, Rampe ja Likka and this relaxing monday evening.

Chamber music

I value playing in small ensembles the most. With different groups there's a variety of music genres and different dynamic between the members. Some memorable events for me have been playing in castles, singing in a choir competition and taking part in international courses.
